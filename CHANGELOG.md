# Changelog

## [1.0.0] - 2026-03-18

### 新增功能
- 系统概览页面：Gateway状态、CPU/内存图表、快捷操作
- Agent管理：列表、详情抽屉、配置文件查看
- Cron任务监控：任务列表、状态统计
- 任务队列：列表、状态筛选、统计
- 任务组与依赖管理：groupId、dependsOn、orderIndex支持
- Tokens监控：消耗统计、趋势图、费用计算
- 失败追踪：Cron/Task失败列表、错误详情、重试
- 女儿国集成：13个Agent配置管理

### 技术栈
- Vue3 + TypeScript
- Vite
- Element Plus
- Pinia
- ECharts

### 部署
- Dockerfile
- docker-compose.yml
- Nginx配置
- 一键部署脚本
